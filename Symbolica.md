#physics

`Symbolica` is a symbolic computation package built with `rust` and usable through `python` API. The [homepage](https://symbolica.io) contains various links and demos.

The pattern matching [documentation](https://symbolica.io/docs/pattern_matching.html) is mainly narrated with python code; for rust examples, the docs.rs pages for [`replace`](https://docs.rs/symbolica/latest/symbolica/atom/trait.AtomCore.html#method.replace) and [`pattern_match`](https://docs.rs/symbolica/latest/symbolica/atom/trait.AtomCore.html#method.pattern_match) are useful.
